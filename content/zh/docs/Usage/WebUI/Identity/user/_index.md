## 查看用户
  * 点击"**筛选**"按钮，输入过滤条件，查询过滤后的用户列表
  * 点击"**列过滤**" 按钮，过滤需要显示在列表也的信息，默认全部显示


{{% imgproc userlist Fit "1000x1000" %}}
用户列表
{{% /imgproc %}}

## 新建用户
点击"**新建**"按钮，输入或选择以下项目:
  * 用户名称
  * 用户昵称
  * 用户头像
  * 用户角色
  * 登录密码 

{{% imgproc usernew Fit "1000x1000" %}}
新建用户
{{% /imgproc %}}

## 用户管理
  * 点击"**编辑**"按钮，修改用户基本信息
  * 点击"**删除**"按钮，删除一个用户
